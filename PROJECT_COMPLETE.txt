If services won't start:
├─ Check logs: docker-compose logs
├─ Verify .env file is correct
├─ Ensure ports are available
└─ Restart services: docker-compose restart

If API is unreachable:
├─ Check backend status: docker-compose logs backend
├─ Verify network: curl http://localhost:5000/health
├─ Check MongoDB: docker-compose logs mongodb
└─ Review firewall rules

If database won't connect:
├─ Verify MONGODB_URI in .env
├─ Check MongoDB is running: docker-compose ps
├─ Review MongoDB logs
└─ Try restarting: docker-compose restart mongodb

If file upload fails:
├─ Check file size (max 10MB)
├─ Verify file type (.pdf, .docx, .txt)
├─ Check uploads directory permissions
└─ Review logs: docker-compose logs backend
